ProfessionalRecorder

ProfessionalRecorder is the first mobile app that I have ever developed. It is a work-in-progress, as I develop both the front end as well as the back end. My goals with this project are to create a highly customizable application that can unlock the full recording potential of your smartphone. I was inspired for this app when I realized how few decent audio recorders there were for android. I set out to create an app that works for every current android phone and helps to deliver the same high quality sound regardless of device. In order to do so, I am utilizing the MediaRecorder API to enhance the recording functionality of this application. Stay tuned for the project to appear on my github and eventually the play store!

  • Code: Github repository
  • Date: September 2016 - Present
